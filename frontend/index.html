The Problem

The gap that costs lives

India's informal caregivers face a crisis that existing technology cannot solve.

🌐

Language Barriers

A caregiver speaks Hindi, Tamil, or Kannada in a panic. Medical staff need structured English. No bridge exists.

📋

Scattered Records

Medication lists live on torn prescription slips and the caregiver's imperfect memory. No single source of truth at the moment it's needed.

⏱️

Zero Guidance

Standard advice lines give generic instructions. No system accounts for the specific patient's drug list, history, and conditions.

🏥

ER Arrives Blind

The hospital ER has zero information about an incoming patient until they physically arrive. Door-to-treatment time suffers.

Solution

One panic call. Life-saving response.

SAHAYAK accepts any input — voice in any Indian language, blurry prescription photos, IoT vitals — and converts them to personalised guidance in under 8 seconds.

    Architecture

    8 GCP Services. Each load-bearing.

    No decorative integrations. Every Google Cloud service performs a function that no other service in the stack can replicate.

    Layer 1 — Input
    🎤

    Cloud Speech-to-Text v2

    Real-time transcription in 12+ Indian languages from panicked, noisy voice calls.

    Layer 1 — Input
    📸

    Cloud Vision API

    OCR for blurry, low-light prescription and discharge paper photos.

    Layer 2 — Intelligence
    🧠

    Vertex AI · Gemini 1.5 Pro

    Single multimodal call fuses all inputs with 1M-token patient history. Temperature 0.1 for clinical consistency.

    Layer 3 — Compute
    ☁️

    Cloud Run

    Scales to zero between incidents. FastAPI async. Deployed in asia-south1 for India latency.

    Layer 4 — Data
    🔥

    Firestore

    Real-time patient profile sync. Offline-capable SDK for 2G resilience.

    Layer 4 — Identity
    📱

    Firebase Authentication

    Phone OTP only. No email or password. Designed for low-digital-literacy caregivers.

    Layer 4 — Interop
    🏥

    Cloud Healthcare API

    Delivers FHIR R4 patient brief to hospital EMR before ambulance arrives.

    Layer 4 — Security
    🔐

    Secret Manager + KMS

    Zero hardcoded credentials. Patient data encrypted at rest with CMEK.
